## 🔧 Optional Configuration

If you want to manually change the Device ID values:

1. Open:

`%LOCALAPPDATA%\Minecraft Device Info\config.txt`

2. The file contains two lines:

- **Line 1:** Your original Device ID.
- **Line 2:** The Device ID the program will spoof with.

Edit the values, save the file, and run `spoofer.exe` again.
